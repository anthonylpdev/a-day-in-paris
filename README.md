# A day in Paris

*Little animated scene with animated SVG.*

![ALP Boilerplate for Three.js](screenshot.jpg)

## Installation
Run this command in the terminal to install dependencies :
```
yarn
```

Compile the code
```
yarn dev
```

Compile the code with a local server
```
yarn watch
```

Compile the code for prod
```
yarn prod
```
